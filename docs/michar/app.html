michar¶

michar [OPTIONS] COMMAND1 [ARGS]... [COMMAND2 [ARGS]...]...

Options

-d, --debug¶

debug logs

--version¶

Show the version and exit.

crawl¶

crawl websites for data

michar crawl [OPTIONS]

Options

-U, --url <url>¶

website to crawl for data

report¶

generate report from results of crawling

michar report [OPTIONS]

Options

-o, --reportType <reporttype>¶

what kind of report to generate

ui¶

Open terminal UI

michar ui [OPTIONS]
